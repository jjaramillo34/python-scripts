DuckDuckGo Image Search API

A powerful REST API for searching and scraping images with advanced filtering options

Version 1.0.0 FastAPI OpenAPI 3.0
Interactive Docs ReDoc OpenAPI Schema

Features

Search Images

Search for images by keywords with advanced filtering options and precise results.

Advanced Filtering

Filter by size, color, type, layout, and license to find exactly what you need.

Region Support

Search in different regions worldwide with customizable regional settings.

Image Validation

Optional URL validation ensures you get reliable and accessible image results.

Pagination

Navigate through multiple pages of results efficiently with built-in pagination.

High Performance

Built with FastAPI for exceptional speed, reliability, and scalability.

API Endpoints

GET /api/search

Search images with query parameters. All parameters except query are optional.

POST /api/search

Search images with JSON request body. Perfect for complex queries with many parameters.

GET /health

Health check endpoint to verify API status and availability.

GET /api/info

Get API information in JSON format including all available endpoints.

Example Usage

GET Request

curl "{{ base_url }}/api/search?query=butterfly&max_results=5"

POST Request

curl -X POST "{{ base_url }}/api/search" \
-H "Content-Type: application/json" \
-d '{"query": "butterfly", "max_results": 5}'

JavaScript Fetch

fetch('{{ base_url }}/api/search?query=butterfly&max_results=5')
.then(response => response.json())
.then(data => console.log(data));
